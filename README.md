# *Multilingual Operations Analytics Dashboard*

## Overview

This portfolio project demonstrates an end-to-end Data Analytics workflow using SQL, Python, Power BI, and Excel. The objective was to analyze multilingual operational data, monitor workflow performance, track key business KPIs, and generate actionable insights through reporting, dashboarding, and exploratory data analysis (EDA).

Real-world multilingual operational workflows inspire the project and showcases the analytical techniques, SQL reporting, ETL processes, and dashboard development approaches commonly used in operational analytics environments.

## Dataset

The portfolio dataset contains 499 sample project records with the following attributes:

* Project ID
* Language
* Client
* Project Status
* Turnaround Time (Hours)
* Word Count
* Delivery Date
* Region

**Note:** The sample dataset was created solely for portfolio demonstration purposes. The analysis methodology, reporting framework, ETL workflow, SQL queries, and dashboard design are based on real-world experience working with multilingual operational datasets containing **50K+ records**.
